Depot

In a few seconds you'll be rolling with a demo version of Depot. Be sure to check out the tutorials and examples included in the install as well as the Depot site depot.snepo.com for updates and new tutorials.

Important! Depot will be installed under your Home folder.

e.g. /Users/YourName/depot

Demo

The demo version of depot allows one connection at a time and allows you to store up to 20 items in a depot directory. It is restricted to non-commercial use but it does not expire.
